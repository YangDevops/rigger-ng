## 个人开发
``` shell

 mkdir ~/devspace ; cd ~/devspace/
 git clone git@github.com:xcodecraft/rigger-ng.git
 cd rigger-ng ;
 setup.sh etc/<platform>.py
 source  ./rigger-ng/myrg.bashrc

 rg help
```
